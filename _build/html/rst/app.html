app.single_employee(id)¶

This will retreive, update and delete the database entries using the id of the entry.

Parameters

  • id (int) – Used to give the id of the entry that you ant to update or delete.

  • name (str) – The name tells us the updates name of the person in the database.

  • salary (float) – The salary tells us about the updated salary of the people in th database.

200 : The request was received and understood and is being processed. 404 : The resource is missing

app.users()¶

This is a data entry API, which get backs the inserted data and also used to insert new data to the database.

Parameters

  • id (int) – The id tells the number of the entry which tells us the specific person in the database.

  • name (str) – The name tells us the name of the person in the database.

  • salary (float) – The salary tells us about the salary of the people in th database.

  • new_name (str) – If you want to add a new entry in the database

  • new_salary (float) – For the new entry you shoud give the new salary

201 : The request was successful and as a result a resource has been created.
